# Bike Sharing Dashboard

## 📌 Introduction
This project provides a Streamlit-based interactive dashboard to analyze the Bike Sharing dataset. The dashboard helps visualize trends in bike rentals based on time, weather, and working conditions.

## 🛠️ Setup & Installation
Follow these steps to set up and run the dashboard:

### 1️⃣ Clone the Repository
```sh
git clone <repository_url>
cd Proyek_Analisis_Data_Akrom/dashboard
```

### 2️⃣ Create and Activate Virtual Environment
#### Windows (PowerShell)
```sh
python -m venv .venv
.venv\Scripts\activate
```
If you encounter a script execution policy issue, run:
```sh
Set-ExecutionPolicy Unrestricted -Scope Process
```
#### macOS/Linux
```sh
python3 -m venv .venv
source .venv/bin/activate
```

### 3️⃣ Install Streamlit dan pustaka lain
```sh
pip install streamlit pandas matplotlib seaborn
```

### 4️⃣ Run the Dashboard
```sh
cd Dashboard
streamlit run dashboard.py
```
